## Setup

1. **Create a `.env` file** and add the following:

    ```env
    MONGODB_URI=...
    PORT=5001
    JWT_SECRET=...
    CLOUDINARY_CLOUD_NAME=...
    CLOUDINARY_API_KEY=...
    CLOUDINARY_API_SECRET=...
    NODE_ENV=development
    ```

2. **Build the App:**
    ```bash
    npm run build
    ```

3. **Start the App:**
    ```bash
    npm start
    ```
